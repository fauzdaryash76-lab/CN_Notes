Unit 4: Network Layer and Transport Layer

The network layer is responsible for routing data packets from the source to the destination across multiple networks. It determines the best path for data transmission and ensures packets reach the correct destination.

Logical addressing uses IP addresses to uniquely identify devices in a network. Internetworking refers to connecting multiple networks together to form a larger network such as the Internet.

Tunneling is a technique used to send data securely across networks by encapsulating packets within other packets. Address mapping converts logical addresses into physical addresses for communication within local networks.

ICMP (Internet Control Message Protocol) is used for error reporting and diagnostics in networking. IGMP (Internet Group Management Protocol) manages multicast group memberships in networks.

Forwarding is the process of sending packets from one router to another toward their destination. Unicast routing protocols determine the best path for sending packets to a single destination, while multicast routing protocols deliver data to multiple receivers simultaneously.

The transport layer ensures reliable data delivery between applications running on different devices. It manages segmentation, reassembly, and process-to-process communication.

UDP (User Datagram Protocol) is a connectionless protocol that provides fast data transmission but does not guarantee reliability. TCP (Transmission Control Protocol) is connection-oriented and ensures reliable and ordered delivery of data. SCTP is another transport protocol that supports multi-stream communication.

Data traffic management controls network traffic to maintain efficient data transmission. Congestion occurs when network resources are overloaded. Congestion control techniques regulate transmission rates to prevent network overload.

Quality of Service (QoS) ensures network services maintain required performance levels such as bandwidth, delay, and reliability. Integrated services and differentiated services are two approaches used to manage QoS in networks.
