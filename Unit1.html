Unit 1: Introduction to Networks

A computer network is a collection of interconnected devices such as computers, servers, printers, and mobile devices that communicate with each other to share resources and information. Networking allows users to exchange data, share hardware resources, and access services over a communication medium.

Networks can be classified into different types such as Local Area Network (LAN), Metropolitan Area Network (MAN), and Wide Area Network (WAN) depending on their geographical coverage. Networking improves communication efficiency and supports distributed computing.

The Internet is the largest global network that connects millions of private, public, academic, business, and government networks. It enables communication and information sharing worldwide through services such as web browsing, email, file transfer, and video communication.

Protocols are defined rules and procedures that allow communication between network devices. They determine how data is formatted, transmitted, and received across the network. Examples include HTTP, FTP, SMTP, and TCP/IP.

The OSI (Open Systems Interconnection) model is a conceptual framework that describes how data communication occurs in a network. It consists of seven layers: Physical, Data Link, Network, Transport, Session, Presentation, and Application.

The TCP/IP suite is the most widely used protocol stack in modern networking. It contains four layers: Network Interface, Internet, Transport, and Application layer. It enables reliable communication and routing across networks.

Addressing in computer networks identifies devices uniquely so data can reach the correct destination. Each device is assigned an IP address that acts as its identification number.

Signals are used to transmit data across communication channels. Analog signals are continuous signals, while digital signals represent data using binary values (0 and 1). Digital transmission is generally more reliable and less affected by noise.
