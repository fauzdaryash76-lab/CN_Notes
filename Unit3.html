Unit 3: LAN Technologies and Wireless Networks

Random access protocols allow multiple devices to access the communication channel without predefined control mechanisms. Devices compete for channel access and transmit data whenever the channel is available. Examples include ALOHA and Carrier Sense Multiple Access (CSMA).

Controlled access protocols regulate how devices share the communication medium by assigning transmission turns to devices. Techniques such as polling and token passing ensure orderly and fair access to the communication channel.

Channelization divides a communication channel into smaller channels so multiple devices can transmit simultaneously. Techniques such as Frequency Division Multiple Access (FDMA), Time Division Multiple Access (TDMA), and Code Division Multiple Access (CDMA) are commonly used.

IEEE standards define specifications for networking technologies. These standards ensure compatibility and interoperability among networking devices. The IEEE 802 family of standards specifically defines LAN and MAN technologies.

Ethernet is the most widely used LAN technology. It supports communication using frames and typically uses a star topology. Fast Ethernet increases the speed up to 100 Mbps, while Gigabit Ethernet provides speeds up to 1000 Mbps.

Wireless LANs enable devices to connect without physical cables using radio frequency signals. Wi-Fi technology allows laptops, smartphones, and other devices to access networks wirelessly in homes, offices, and public spaces.

Connecting LANs requires networking devices such as bridges, switches, and routers. Backbone networks provide high-speed connections that link multiple LAN segments together.

Virtual LANs (VLANs) allow administrators to logically divide a physical network into multiple virtual networks. VLANs improve security, performance, and traffic management.

Wireless WAN technologies support long-distance wireless communication using cellular networks and satellite communication systems. Technologies such as SONET, Frame Relay, and ATM are used for high-speed data transmission across wide area networks.
