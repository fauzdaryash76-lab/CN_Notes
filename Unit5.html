Unit 5: Application Layer and Network Security

The application layer is the top layer of the network architecture and provides services directly to users and applications. It enables network applications such as email, web browsing, file transfer, and remote login.

The Domain Name System (DNS) is a hierarchical system that translates human-readable domain names into IP addresses. It allows users to access websites using easy-to-remember names instead of numeric addresses.

Electronic mail is a widely used Internet service that allows users to send and receive messages electronically. Protocols such as SMTP, POP3, and IMAP are used for email communication.

File Transfer Protocol (FTP) is used to transfer files between computers over a network. It allows users to upload and download files from remote servers.

The World Wide Web (WWW) provides access to web pages and online resources through web browsers. HTTP is the protocol used for transferring web pages between servers and clients.

SNMP (Simple Network Management Protocol) is used for monitoring and managing network devices such as routers and switches. Multimedia applications allow transmission of audio, video, and interactive media over networks.

Telnet is a protocol that allows remote login to another computer over a network. Bluetooth is a short-range wireless communication technology used for connecting devices such as smartphones, headphones, and laptops.

Firewalls are security systems that monitor and control incoming and outgoing network traffic based on predefined security rules. They protect networks from unauthorized access and cyber threats.

Cryptography is the science of securing information by converting data into an unreadable format using encryption techniques. Network security involves protecting network resources, data, and communication systems from attacks and unauthorized access.
